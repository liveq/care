## 1. 프로젝트 개요

### 1.1 서비스 목적
- 디지털 성범죄 피해자, 개인정보 유출 피해자, 온라인 평판 관리가 필요한 개인/기업을 위한 전문 서비스
- AI 기술 + 정보보호 전문성을 활용한 차별화된 디지털 프라이버시 관리
- 국내외 포털/SNS/커뮤니티에 유포된 정보의 삭제 및 지속적 관리

### 1.2 핵심 차별화 포인트
- **정보보호학과 출신 + 디지털 장의사 자격증 보유**
- **AI 크롤링 기술을 활용한 신속한 유포 범위 파악**
- **방송통신심의위원회 및 국내외 플랫폼과의 협력 체계**
- **100% 비밀 보장 - 후기/사례 공개 일절 금지**

### 1.3 서비스 위치
- **메인 도메인**: diora.co.kr
- **서비스 서브도메인**: care.diora.co.kr
- **저장소 구조**: diora.co.kr 저장소 내 `/care` 폴더
- **배포 방식**: GitHub Pages 자동 배포

## 2. 서비스 정의

### 2.1 공식 서비스명 (3단 표현)
1. **디지털 프라이버시 관리** (Digital Privacy Management)
2. **온라인 평판 보호** (Online Reputation Protection)  
3. **디지털 족적 관리** (Digital Footprint Management)

### 2.2 서비스 범위

#### 2.2.1 국내 플랫폼
- 네이버, 다음, 구글 코리아
- 인스타그램, 페이스북, 트위터(X)
- 유튜브, 틱톡
- 웹하드 (웹디스크, 파일조, 피디박스 등)
- 커뮤니티 (디시인사이드, 에펨코리아, 보배드림, 뽐뿌 등)
- 취업 플랫폼 (잡플래닛, 블라인드)
- 중고거래 (당근마켓, 번개장터)

#### 2.2.2 해외 플랫폼
- Google (미국)
- Instagram, Facebook (Meta)
- Twitter (X Corp.)
- Reddit, Tumblr
- Pornhub, Xvideos 등 성인 사이트

### 2.3 주요 의뢰 유형
- **디지털 성범죄 피해**: 불법 촬영물, 딥페이크, 리벤지 포르노
- **개인정보 유출**: 주민등록번호, 연락처, 주소 등
- **악성 게시물**: 명예훼손, 허위사실 유포, 악플
- **과거 기록 삭제**: 범죄 기록, 불명예 기사, 오래된 SNS 게시물
- **기업 평판 관리**: 허위 리뷰, 근거 없는 비방

### 5.7 FAQ

```
[H2] 자주 묻는 질문

Q1. 정말 삭제가 가능한가요?
A: 네, 가능합니다. 다만 100% 완벽한 삭제는 보장할 수 없습니다. 개인이 다운로드해서 소장한 자료까지는 통제가 불가능하기 때문입니다. 저희는 온라인에서 검색되고 유포되는 경로를 최대한 차단합니다.

Q2. 얼마나 걸리나요?
A: 국내 사이트는 1~3일, 해외 사이트는 2주~1개월 소요됩니다. 플랫폼의 응답 속도와 법적 절차에 따라 달라집니다.

Q3. 비밀이 보장되나요?
A: 모든 의뢰는 100% 비밀로 처리되며, 어떠한 사례나 후기도 공개하지 않습니다. 고객이 서비스를 이용했다는 사실조차 남기지 않는 것이 저희의 원칙입니다.

Q4. 착수금은 왜 환불이 안 되나요?
A: 착수금 납부 시점부터 AI 크롤링, 수작업 조사, 법적 검토 등의 작업이 시작되어 실제 인건비와 리소스가 투입되기 때문입니다.

Q5. 불법 게시물이 아니어도 삭제 가능한가요?
A: 불법 게시물이 아니더라도 명예훼손, 사생활 침해, 초상권 침해 등의 사유가 있다면 삭제 요청이 가능합니다. 다만 공익적 게시물(언론 보도 등)은 삭제가 제한될 수 있습니다.

Q6. 해외 사이트는 어떻게 삭제하나요?
A: 각 플랫폼의 공식 신고 채널을 통해 영문으로 삭제 근거를 작성하여 제출합니다. 필요시 법적 문서(위임장, 증명서)를 함께 제출합니다.

Q7. 삭제 후 다시 올라오면 어떻게 하나요?
A: 단건 삭제의 경우 재업로드는 별도 비용이 발생하지만, 스탠다드/프리미엄 패키지는 모니터링이 포함되어 재업로드 시 즉시 대응합니다.

Q8. 경찰 신고도 같이 해야 하나요?
A: 디지털 성범죄의 경우 경찰 신고와 병행하는 것이 효과적입니다. 신고 접수증이 있으면 플랫폼에서 더 신속하게 처리됩니다.

Q9. 위임장은 왜 필요한가요?
A: 개인정보보호법에 따라 본인이 아닌 제3자가 삭제를 요청하려면 법적 위임장이 필요합니다. 이는 합법적인 대리 업무 수행을 위한 필수 절차입니다.

Q10. 어떤 정보를 제공해야 하나요?
A: 삭제를 원하는 게시물의 URL, 파일명, 키워드 등을 제공해 주시면 됩니다. 민감한 자료는 직접 보내지 않으셔도 되며, 설명만으로도 진행 가능합니다.
```

```javascript
function toggleLanguage(lang) {
  const elements = document.querySelectorAll('[data-i18n]');
  elements.forEach(el => {
    const key = el.getAttribute('data-i18n');
    el.textContent = getNestedTranslation(translations[lang], key);
  });
  localStorage.setItem('preferredLang', lang);
  updateURL(lang);
}

// URL 업데이트
function updateURL(lang) {
  if (lang === 'en') {
    window.location.href = '/care/en';
  } else {
    window.location.href = '/care';
  }
}
```

## 11. 위임장 양식 법적 검토

### 11.1 개인정보보호법 근거

```
「개인정보 보호법」 제38조 (권리행사의 방법 및 절차)
① 정보주체는 제35조부터 제37조까지의 규정에 따른 권리 행사를 
개인정보처리자에게 서면, 전화, 전자우편, 모사전송(FAX) 등으로 할 수 있으며, 
개인정보처리자는 이에 대하여 지체 없이 조치하여야 한다.

② 정보주체는 제1항에 따른 권리 행사를 정보주체의 
법정대리인이나 위임을 받은 자 등 대리인을 통하여 할 수 있다. 
이 경우 "개인정보 처리 방법에 관한 고시" 별지 제11호서식에 따른 
위임장을 제출하여야 한다.
```

### 11.2 수정 사항 없음

제공된 위임장 양식은 개인정보보호법 시행규칙의 표준 양식을 준수하며, 
다음 사항이 포함되어 법적으로 유효합니다:

✓ 위임자 인적사항
✓ 수임자 사업자 정보
✓ 위임 내용 명시
✓ 위임 기간
✓ 서명 및 날인
✓ 신분증 사본 첨부

### 11.3 추가 권장사항

```
1. 위임장 유효기간을 계약 종료일까지로 명시
2. 수임자의 개인정보 처리 제한 조항 포함 (완료)
```
